Var

Es una forma de declrar varibales en JS, recordemos que al ser global no es recomendado.

Ejemplo

var nombre = "Juan";

var edad = 23;

var apellido = "Pérez";

Importante: Recordemos que podemos cambiar el valor de la variable sin importar si ésta previamente tenía otro tipo

let

Es una forma de declrar varibales en JS, ahora es el reemplazo de var ya que funciona en un ámbito determinado y no es global.

Ejemplo

let nombre = "Juan";

let edad = 23;

let apellido = "Pérez";

Importante: Recordemos que podemos cambiar el valor de la variable sin importar si ésta previamente tenía otro tipo

const

Es una forma de declrar varibales en JS, pero estas son inmutables, es decir, el valor asignado no puede ser cambiado.

Ejemplo

const nombre = "Juan";

const edad = 23;

const apellido = "Pérez";

Importante: Recordemos que podemos cambiar el valor de la variable sin importar si ésta previamente tenía otro tipo

if

Es una forma de declrar condiciones en JS, y si se cumple la condición se puede ejecutar algo.

Ejemplo

const edad = 18;

if (edad >= 18) {;

    console.log ("Es mayor de edad");

}

Importante: Recordemos que podemos cambiar el valor de la variable sin importar si ésta previamente tenía otro tipo

Else

Es una forma de declrar condiciones en JS, en caso la condición del if no se cumpla, automáticamente se ejecuta lo que esta dentro del else.

Ejemplo

const edad = 18;

if (edad >= 18) {;

    console.log ("Es mayor de edad");

} else {

    console.log ("Es menor de edad");

}

Importante: Evitemos el uso del else, sólo cuando es necesario.

operadores

Vamos a revisare los distintos operadores que podemos usar en los if y else if

Ejemplo

&& : significa Y

|| : significa o

! : significa negación

=== : para comparar si dos valores son iguales

> >= < <= : para comparaciones entre 2 valores

!== : para saber si 2 valores son distintos

operador ternario

Un operador ternario es un if de una linea, ojo sólo podemos ejecutar una linea de codigo dentro de la condición del operador ternario.

Ejemplo

const nombre = "Juan";

nombre === "Juan" ? console.log("Lo encontramos") : ("No lo encontramos :(")

Arreglos

Un arreglo es una colección de daots, recuerden que los arreglos pueden contener cualquier tipo de datos.

  • String
  • Number
  • Boolean
  • Object
  • function

Ejemplo

const arreglo = [`Pepe`,2,true, 90.4,5]

console.log(arreglo);

Importante: Los arreglos se empiezan a contar desde el índice 0, es decir que el primer elemento tiene como indice 0.

lenght

Es la función la cual nos permite saber la cantida de objetos en un arreglo.

Accediendo a un arreglo

Para acceder a un arreglo podemos usar el índice del elemento que queremos acceder.

Ejemplo

const arreglo = [`Pepe`,2,true, 90.4,5]

-Accedemos al primer elemento-

console.log(arreglo[0])

-Accedemos al segundo elemento-

console.log(arreglo[1])

Push

Es la función que nos pernmite agregar un objeto al final del arreglo.

Ejemplo

const arreglo = [`Pepe`,2,true, 90.4,5]

arreglo.push(true);

arreglo.push({nombre: "juana", edad: 18})

console.log(arreglo);

Function

Una función es una agrupación de codigo que se puede ejecutar varias veces, es decir m una funcin puede recibir paramtros y retornar valores. Es para retuilizar codigo.

Importante: Si vemos que una accion se repite mas de una vez lo convertimos a una function

Ejemplo

function saludar(nombre) {

   console.log(`Hola ${nombre}`);

}

// Es la forma es la que llamos a la funcion y le pasamos como parametro el nombre Juan

saludar("Juan");

Function Anonima

Una function anonima basicamente no tiene nombre, esta se suele guardar en una variable

Ejemplo

const saludar = function(nombre) {

   console.log(`Hola ${nombre}`);

}

saludar("Juan");

Ejemplo 2

const saludar = (nombre) => {

   console.log(`Hola ${nombre}`);

}

saludar("Juan");

Arrow Function

Un arrow function es una funcion de tipo la flecha la cual cumple con casi todo lo que hacer un function normal, sin embargo la diferencia entre una arrow funtion y una function normal es que en un arrow function el this no existe.

Ejemplo

const saludar = (nombre) => {

   console.log(`Hola ${nombre}`);

}

saludar("Juan");

Arrow Function Inline

basicamente es un arrow que solo ejecuta un linea de codigo, ojo el return no hace falta porque esta implicito

Ejemplo

const saludar = nombre => console.log(`Hola ${nombre}`);

saludar("Juan");

Importante: Si tenemos solo un parametro en la funcion no hace falta poner los parentesis

Function in function

El concepto de invocar una función dentor de otra es algo ormal y muy usado, se pueden usar n funciones dentro de una función, de hecho es lo ideal tratar de usar funciones para poder hacer que nuestro código esté distribuido.

Ejemplo

const calcularMayorDeEdad =(edad) => {

   const esMayor = esMayor (edad);

   return esMayor

}

//función es mayor

const esMayor (edad) => edad >= 18 ? true : false

Leer Objetos

La forma de acceder a la propiedad de un objeto es por el . ojo si tienen un objeto dentro de otro objeto se puede usar el . para navegar entre el objeto.

Ejemplo

const persona => {

    nombre: "Juancito",

    apellido: "Pérez",

    123: 34,

    domicilio: "Av. Siempreviva 123",

}

// Imprimiendo la propiedad nombre

console.log(pesona.nombre);

console.log(pesona["123"]);

For

El bucle es una estructura repetitiva, es decir hacer algo mcuhas veces y además los bucles son usador para leer arrayts, porque un bucle nos va a permitir leer cada elemento de forma independiente.

Ejemplo

for (let i = 0; i < 10; i++) {

   console.log(i)

}

While

Es otro tipo de bucle que sirve para iterar arrays pero esta atado a una condición o contador.

Ejemplo

let contador = 0;{

while (contador < 10)

    console.log(contador);

    contador++;

}

Importante No se olviden de aumentar el contador porque si no, nuestras computadoras van a explotar.

Map

Es un bucle que permite iterar array pero con la peculiaridad de que este retorna un array.

Ejemplo

const alumnos = ["pepe", "sutano","mengano"]{

alumnos.map((alumno,index) => console.log(index, alumno))

Recomendación Cuando hagamos un maop y este nos solicite el parametro de elemento pdos usar el singular del arreglo.

alumnos => alumno

mascotas => mascota

computadoras => computadoras

For Each

Sirve para iterar datos, sin embargo, a diferencia del map, este no retorna nada. Es decir, si yo guardo en una variable al foreach el resultado será undefined.

Ejemplo

const alumnos = ["pepe", "sutano","mengano"]{

alumnos.foreach((alumno,index) => console.log(index, alumno))

Filter

Es el hermano de map, porque filter es usado para poder filtrar en base a una condición y este retorna un array.

Ejemplo

const alumnos = ["pepe", "sutano","mengano"]{

const alumnoFilter = alumnos.filter((alumno,index) => alumno === "Pepe"))

//Va a retornar ["Pepe"]

Find

Es un iterador de bucles pero sólo retorna un resultado, y este resultado será el primero que encuentre, sin importar que halla más iguales en el array u objeto. Es decir, si en una lista tengo dos o más resultados, sólo va a tomar al primero y ese será el que devuelva.

Ejemplo

const alumnos = ["pepe", "sutano","mengano"]{

const alumnoFind = alumnos.find((alumno,index) => alumno === "Pepe"))

//Va a retornar con Pepe pero como string, porque sólo será un dato.

FindIndex

Sirve para poder encontrar la posición de un elemento en un array. Es decir retorna el índice el elemento encontrado.

Ejemplo

const alumnos = ["pepe", "sutano","mengano"]{

const alumnoFindIndex = alumnos.findIndex((alumno,index) => alumno === "Pepe"))

//Va a retornar 0 porque es el índice que tiene Pepe.
